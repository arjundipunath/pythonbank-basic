## üè¶ PythonBank - Basic Console Banking System

This is a **simple console-based banking application** built with Python. It serves as a project for beginners and intermediate developers to practice core programming concepts like **Object-Oriented Programming (OOP)**, control flow, functions, and data manipulation.

### ‚ú® Features

The application provides a command-line interface (CLI) for users to perform fundamental banking operations:

  * **Create Account:** Register a new customer with an initial deposit.
  * **Deposit Funds:** Add money to an existing account.
  * **Withdraw Funds:** Remove money from an existing account, with a check for sufficient balance.
  * **Check Balance:** View the current balance of an account.
  * **Simple Data Storage:** Account data is managed in memory (or a simple file/dictionary structure, depending on the implementation).

### üõ†Ô∏è Getting Started

Follow these instructions to get a copy of the project up and running on your local machine.

#### Prerequisites

  * **Python 3.x** (latest version recommended)

#### Installation and Setup

1.  **Clone the Repository:**
    Use `git` to download the project files to your local machine.

    ```bash
    git clone https://github.com/arjundipunath/pythonbank-basic.git
    ```

2.  **Navigate to the Project Directory:**

    ```bash
    cd pythonbank-basic
    ```

3.  **Run the Application:**
    Execute the main script to start the banking system interface.

    ```bash
    python main.py
    ```

    *(Note: The main script may be named `bank.py` or similar; please verify the filename in the repository if `main.py` does not work.)*

Once the application is running, you will be presented with a simple menu in your console:
